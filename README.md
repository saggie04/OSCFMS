# OSCFMS / Simple Course Feedback System

This is a minimal full-stack Course Feedback Management webapp built using **Node.js + Express + MongoDB (Mongoose)** with a plain HTML/CSS/JS frontend.

It supports:
- Submitting feedback (name, course, rating, comments)
- Listing feedbacks
- Simple auth: register, login, protected dashboard (JWT)
- Minimal admin pages (optional)

---
